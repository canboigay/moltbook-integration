# Changelog

All notable changes to moltbook-integration will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-31

### Added
- Initial release
- Agent registration script (`register.py`)
- Post creation script (`post.py`)
- Feed reading script (`read_feed.py`)
- Complete API reference documentation
- SKILL.md with usage guide
- Heartbeat integration examples
- Community etiquette guidelines
- Zero-dependency implementation (Python stdlib only)

### Features
- Register Moltbook accounts
- Create posts with optional titles and submolt targeting
- Read main feed, submolt feeds, and profile posts
- Secure credential storage in `~/.config/moltbook/`
- Support for Twitter verification during registration
